# MERN Note-Taking App 

A full-stack note-taking web application built using the MERN stack, with authentication and real-time state management using Redux-Saga.

---

#  Tech Stack

- Frontend: React, Redux Toolkit, Redux-Saga, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB (via MongoDB Atlas)
- Auth: JWT (JSON Web Tokens) for secure login/signup

---

# Features

-  User Signup & Login (JWT-based authentication)
-  Add Note
-  Edit/Update Note
-  Delete Note
-  Fully connected frontend & backend
- State management with Redux-Saga
